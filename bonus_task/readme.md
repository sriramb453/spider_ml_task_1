Bonus Task: Multi-Paper Comparison Feature

This directory documents the advanced multi-paper comparison capability implemented as part of the induction tasks.

Architecture & Strategy
Instead of building a separate, disconnected script, the comparison logic has been seamlessly integrated directly into the core production backend application located at:
`applied_ml_domain/chatbot_code/main.py`

Advanced Retrieval Routing: When a user submits a query containing comparative keywords (e.g., compare, contrast, difference, versus, vs), the backend automatically triggers an alternative retrieval track.

Max Marginal Relevance (MMR): Rather than standard similarity search (which can get trapped pulling contiguous blocks from a single document), the engine uses MMR (`vector_db.max_marginal_relevance_search`) with a high diversity factor (`fetch_k=15`, `k=5`). This guarantees that context blocks are extracted from highly diverse segments across multiple distinct research papers simultaneously.

Structured Context Structuring: Extracted fragments are grouped and formatted with explicit metadata tags outlining their source files before being passed to the local Large Language Model.

Directory Contents
`results/comparison_evaluation.md`: A comprehensive production log demonstrating a live, real-world evaluation executed on the local engine. It tracks the test query ("Compare the core architectures and training data sizes between the Attention Is All You Need paper and the LLaMA paper"), details the multi-source context extraction behavior, and displays the exact text output generated by the local LLaMA 3.2 model.

 How to Experience the Feature

You do not need to run separate command-line test scripts. The comparison feature is fully active inside the main application interface:
1. Ensure your local Ollama instance is active (`ollama run llama3.2`).
2. Boot up the unified FastAPI server from the root repository directory:
   `python3 -m uvicorn applied_ml_domain.chatbot_code.main:app --reload`
3. Launch the custom web UI (`applied_ml_domain/chatbot_code/index.html`).
4. Type a cross-paper query like "Compare GPT-3 vs LLaMA parameters" or "Contrast the training datasets between the Transformer and BERT papers", and watch the system dynamically cross-reference multiple source sheets in real-time.
